# Tic Tac Toe Game

Welcome to Tic Tac Toe! This is a simple web-based game where two players take turns marking spaces on a 3x3 grid to get three of their symbols (either "X" or "O") in a row, column, or diagonal.

## How to Play

1. **Enter Player Names**: Start the game by entering the names of the two players in the input fields provided.
2. **Gameplay**: Players take turns clicking on empty squares to place their symbol. Player 1 starts with "X", and Player 2 follows with "O".
3. **Winning**: The game ends when one player gets three of their symbols in a row, column, or diagonal, or if all squares are filled without a winner (resulting in a draw).
4. **Restart**: After a game ends, you can restart the game by refreshing.

## Technologies Used

This game is implemented using the following technologies:

- **HTML**: Provides the structure and layout of the game board.
- **CSS**: Styles the appearance of the game board and symbols.
- **JavaScript**: Implements the game logic, including handling player turns, checking for a winner, and updating the game board dynamically.
